안녕하세요! 멋진 프론트앤드 개발자를 꿈꾸는 돌맹이 입니다

항상 새로운 것에 도전하고, 노력하는 열정이 넘치게 개발 공부를 해왓습니다

자기소개

2021년 1월 부터 제대로 개발 공부를 시작 하였고, 프론트앤드 개발자에게 필요한 HTML, CSS, JS, React을 공부 해왔습니다. 항상 새로운 기술에 관심을 많이 갖고 있으면 어떻게 하면 좀 더효율적인 코드를 짤수 있을까 고민 해왔습니다. 열정 넘치는 개발자의 모습을 보여드리겠습니다

  • HTML

    기본적인 마크업, DOM 렌더링 과정을 숙지하였습니다

  • CSS

    기본적인 디자인, flex-box, 에니메이션을 숙지 하였습니다

  • JS

    반복문, 스코프, 호이스팅등 개념을 숙지 하였고, DOM과 호환하는 이해를 하였습니다

  • React

    React에 대한 Hooks, redux 상태관리등 숙지 하였습니다
